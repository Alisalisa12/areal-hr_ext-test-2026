# Инструкция для менеджера

Менеджер системы отвечает за ведение кадрового учета сотрудников.

## Работа с сотрудниками

Это основной раздел системы для ведения учета.

1. **Создание карточки сотрудника**:
   - Перейдите в раздел **"Сотрудники"**.
   - Нажмите кнопку **"Добавить сотрудника"**.
   - Заполните личные данные (ФИО, дата рождения, паспортные данные и адрес).
2. **Редактирование данных и увольнение**:
   - Для изменения информации нажмите на иконку карандаша в строке нужного сотрудника.
   - Вы можете уволить сотрудника используя соответсвующую иконку в графе **"Действия"**.

## Организации, отделы и должности

- **Организации**: Создание компаний.
- **Отделы**: При создании отдела необходимо выбрать организацию, к которой он относится или отделу.
- **Должности**: Должность привязывается к конкретному отделу.

## Поиск и фильтрация

Система позволяет быстро находить нужную информацию в базе данных.

- **Поиск**: Используйте строку поиска для мгновенного нахождения информации.
- **Фильтры**: Вы можете отфильтровать список.
- **Сортировка**: Нажимайте на заголовки столбцов таблицы, чтобы выполнить сортировку.

## Хранение документов

Для сотрудников менеджер может загружать необходимые документы (паспорт, инн и тд.).

- Все файлы надежно хранятся в облачном хранилище **MinIO**.

## HR операции

Менеджер может выполянть следующие операции:

- Прием на работу
- Перевод
- Изменение зарплаты
- Увольнение

А также просматривать историю операций.
